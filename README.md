ShoeBattle - PWA Prototype
-------------------------
Descripción:
- Registra tus zapatos y cuenta cuántas veces los usas.
- Battle Mode muestra ranking de zapatos más usados.
- Guarda los datos localmente en el móvil (iPhone/Android).
- Instalación: desde Safari en iPhone, menú -> "Add to Home Screen".

Archivos:
- index.html : Página principal
- style.css  : Estilos
- app.js     : Lógica de zapatos y ranking
- manifest.json : PWA manifest
- /icons     : iconos placeholder

Instrucciones:
1) Subir archivos a Netlify Drop (https://app.netlify.com/drop) o GitHub Pages.
2) Abrir la URL en Safari en iPhone.
3) "Add to Home Screen" para instalar.
4) Añadir zapatos y contar usos con el botón '+1'.
